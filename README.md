# Paroles

Minimal CLI for grabing lyrics from `genius.com` 

## Installation

```bash
gem install nokogiri
```

```ruby
# Necessary libraries:
require 'json'
require 'net/http'
require 'nokogiri'
require 'uri'
```

## Usage

```bash
(ruby) paroles <query>

" for example:
paroles get lucky
```
